# BS450G3

4U通用存储型服务器，配置双插槽第三代Intel® Xeon® Scalable处理器；支持16个ECC DDR4-3200内存插槽和2个Intel® Optane™持久内存专用插槽；具备36个热插拔SAS3/SATA3硬盘槽（包括4个混合NVMe槽）、2个后置SATA槽和1个SATA/NVMe M.2插槽，提供卓越的数据存储能力

## 详细规格

| 参数 | 规格 |
|------|------|
| Form Factor | 4U Rackmount |
| Processor | 3rd Gen Intel Xeon Scalable Processor;Support CPU TDP Up to 270W |
| Socket | 2 Socket P+ (LGA-4189) |
| Chipset | Intel C621A |
| Memory | Up to 16 DIMM slots<br>Up to 4TB: 16x 256GB DRAM |
| Memory Type | Support 3200MHz ECC DDR4 RDIMM or LRDIMM |
| Storage | 36x 3.5" hot-swap SATA3/SAS3 drive bays |
| Storage | 2x 2.5" SATA drive bays |
| Storage | 1 M.2 NVMe/SATA |
| PCIe Slots | 3 PCIe 4.0 x16 LP slots<br>1 PCIe 4.0 x8 LP slots |
| Network | 2x 10GbE BaseT |
| PSU | 2x 1600W Redundant Titanium Power SupplyLevel Power Supply Unit |
| Dimensions | Height: 178 mm (7")<br>Width: 437 mm (17.2")<br>Depth: 698 mm (27.5") |
