# BS450G3-S

## 产品介绍

4U通用存储型服务器，配置双插槽第三代Intel® Xeon® Scalable处理器；支持16个ECC DDR4-3200内存插槽和2个Intel® Optane™持久内存专用插槽；具备36个热插拔SAS3/SATA3硬盘槽（包括4个混合NVMe槽）、2个后置SATA槽和1个SATA/NVMe M.2插槽，提供卓越的数据存储能力

## 详细规格

| 规格项目        | 详细说明                                                                             |
| --------------- | ------------------------------------------------------------------------------------ |
| **外形规格**    | 4U机架式                                                                             |
| **处理器**      | 第3代Intel Xeon Scalable处理器；支持CPU TDP最高270W                                  |
| **插槽**        | 2 Socket P+ (LGA-4189)                                                               |
| **芯片组**      | Intel C621A                                                                          |
| **内存**        | 16个ECC DDR4-3200内存插槽                                                            |
| **内存类型**    | 支持3200MHz ECC DDR4 RDIMM或LRDIMM                                                   |
| **Optane**      | 2个Intel® Optane™持久内存专用插槽                                                    |
| **存储**        | 36个热插拔SAS3/SATA3硬盘槽（包括4个混合NVMe槽）、2个后置SATA槽和1个SATA/NVMe M.2插槽 |
| **存储 (NVMe)** | 1个M.2 NVMe                                                                          |
| **PCIe插槽**    | 3个PCIe 4.0 x16 LP插槽、1个PCIe 4.0 x8 LP插槽                                        |
| **网络**        | 2x10GbE BaseT网络接口                                                                |
| **PSU**         | 2x1600W冗余钛金级电源                                                                |
| **尺寸**        | 高度：178mm (7英寸) 宽度：437mm (17.2英寸) 深度：698mm (27.5英寸)                    |

## 产品特点

- 高性能计算：搭载第3代Intel Xeon Scalable处理器，支持CPU TDP最高达270W
- 大容量内存：支持16个ECC DDR4-3200内存插槽，支持最大4TB内存
- 灵活存储：36个热插拔SAS3/SATA3硬盘槽，支持混合NVMe配置
- 高速扩展：提供多个PCIe 4.0插槽，满足不同扩展需求
- 冗余电源：配备双1600W冗余钛金级电源，确保稳定运行
- 4U机架式设计：适合数据中心部署，优化空间利用
