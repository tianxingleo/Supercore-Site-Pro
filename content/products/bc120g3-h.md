# BC120G3-H

## 产品描述

1U通用计算型服务器，搭载双路3rd Gen Intel® Xeon® Scalable系列处理器，可扩展至32个DIMM插槽和8TB ECC DDR4 DRAM；最多支持4个PCIe 4.0插槽，提供高速数据传输和扩展性；最多提供12个2.5英寸硬盘插槽，满足大容量存储需求；配备最多两个1200瓦冗余钛金级PSU，确保稳定的电源供应，提高系统可靠性

## 详细规格

| 参数项        | 详细参数                                                                                 |
| ------------- | ---------------------------------------------------------------------------------------- |
| **外形尺寸**  | 1U机架式                                                                                 |
| **处理器**    | 3rd Gen Intel Xeon Scalable Processor; CPU TDP最高270W                                   |
| **插槽**      | 2 Socket P+(LGA-4677)                                                                    |
| **芯片组**    | Intel C621A                                                                              |
| **内存**      | 最多32个DIMM插槽<br>最高8TB：32x256GB DRAM/PMem                                          |

| **存储**      | 12x 2.5英寸热插拔NVMe/SATA/SAS 硬盘托架                                                  |
| **SATA**      | 支持SATA3(6Gb/s) RAID 0/1/5/10 与 Intel VROC                                             |
| **NVMe**      | 支持NVMe RAID 0/1/5/10 与 Intel VROC                                                     |

| **网络**      | O1: 无<br>O2: 2x10GbE Base-T(X710-AT2)<br>O3: 2x10GbE Base-T 和 2x10GbE SFP+(X710-TM4)   |

## 产品特点

- 搭载双路3rd Gen Intel Xeon Scalable系列处理器，支持高达270W TDP
- 可扩展至32个DIMM插槽，最大支持8TB内存
- 支持3200MHz ECC DDR4 RDIMM或LRDIMM内存
- 支持Intel Optane Persistent Memory 200系列
- 提供最多4个PCIe 4.0插槽，包括1个内部LP插槽、1个LP插槽和2个FHHL插槽
- 支持多达12个2.5英寸热插拔NVMe/SATA/SAS硬盘位
- 支持SATA3(6Gb/s)和NVMe的RAID 0/1/5/10（通过Intel VROC）
- 多种网络配置选项：无网络、2x10GbE Base-T或2x10GbE Base-T + 2x10GbE SFP+
- 配备2x1200W冗余钛金级电源，确保系统稳定运行
- 1U机架式设计，尺寸为43mm x 437mm x 739mm，节省数据中心空间
