# BC120G3

1U通用计算型服务器，搭载双路3rd Gen Intel® Xeon® Scalable系列处理器，Intel C621A芯片组，支持PCIe 4.0协议，提供最大16根3200MT/s DDR4内存插槽，前置支持4块3.5英寸SATA/SAS或8块2.5英寸SATA/SAS+2块2.5英寸NVMe/SATA/SAS热插拔硬盘，满足计算需求。

## 详细规格

| 规格项目       | 详细参数                                                                                                                |
| -------------- | ----------------------------------------------------------------------------------------------------------------------- |
| **外形尺寸**   | 1U机架式                                                                                                                |
| **处理器**     | 3rd Gen Intel Xeon Scalable Processor; CPU TDP最高270W                                                                  |
| **插槽**       | 2 Socket P+(LGA-4189)                                                                                                   |
| **芯片组**     | Intel C621A                                                                                                             |
| **内存**       | 最多16个DIMM插槽，最高4TB：16x256GB DRAM                                                                                |
| **内存类型**   | 支持3200MHz ECC DDR4 RDIMM 或 LRDIMM                                                                                    |
| **存储**       | O1: 4x3.5英寸热插拔SATA/SAS硬盘托架<br>O2: 8x2.5英寸热插拔SATA/SAS硬盘托架<br>O3: 10x2.5英寸热插拔NVMe/SATA/SAS硬盘托架 |
| **存储 (M.2)** | 2个M.2 NVMe                                                                                                             |
| **SATA**       | 支持SATA3(6Gb/s) RAID 0/1/5/10 与 Intel VROC                                                                            |
| **NVMe**       | 支持NVMe RAID 0/1/5/10 与 Intel VROC                                                                                    |
| **PCIe插槽**   | 2个PCIe 4.0 x16 FHHL插槽                                                                                                |
| **网络**       | 2个PCIe 4.0 x16插槽用于AIOM(OCP 3.0)支持NC-SI                                                                           |
| **电源**       | 2x860W冗余白金级电源供应单元                                                                                            |
| **尺寸**       | 高度:43mm (1.7") 宽度:437mm (17.2") 深度:650mm (25.6")                                                                  |

## 产品特点

- **高性能处理器**：搭载双路第三代英特尔至强可扩展系列处理器，支持最高270W TDP，提供强大的计算能力
- **高速内存支持**：配备16个DIMM插槽，支持3200MHz ECC DDR4 RDIMM或LRDIMM，最高支持4TB内存
- **灵活存储配置**：提供三种存储配置选项，满足不同场景的存储需求
  - 选项1：4块3.5英寸热插拔SATA/SAS硬盘
  - 选项2：8块2.5英寸热插拔SATA/SAS硬盘
  - 选项3：10块2.5英寸热插拔NVMe/SATA/SAS硬盘
- **RAID支持**：支持SATA3(6Gb/s)和NVMe的RAID 0/1/5/10，配合Intel VROC技术提供数据保护
- **PCIe 4.0支持**：支持PCIe 4.0协议，提供2个PCIe 4.0 x16 FHHL插槽
- **网络扩展**：提供2个PCIe 4.0 x16插槽用于AIOM(OCP 3.0)网卡，支持NC-SI
- **冗余电源**：配备2x860W冗余白金级电源，确保系统稳定运行
- **紧凑设计**：1U机架式设计，尺寸为43mm x 437mm x 650mm，节省数据中心空间
