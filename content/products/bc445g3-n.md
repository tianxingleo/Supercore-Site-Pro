# BC445G3-N

## 产品描述

4U高性能计算型GPU服务器，融合了NVIDIA® HGX™ A100 8-GPU，采用NVIDIA® NVLINK™ v3.0 + NVIDIA® NVSwitch™实现GPU之间高效通信；双AMD EPYC™ 7003/7002系列处理器；支持至多10个U.2 NVMe 2.5英寸驱动器；提供丰富的PCIe 4.0接口扩展，满足高性能模型计算需求。

## 详细规格

| 规格项       | 详细参数                                |
| ------------ | --------------------------------------- |
| **外形尺寸** | 4U机架式                                |
| **处理器**   | 双AMD EPYC™ 7003/7002系列处理器         |
| **GPU支持**  | NVIDIA® HGX™ A100 8-GPU                 |
| **内存**     | 8TB ECC DDR4内存                        |
| **存储**     | 支持至多10个U.2 NVMe 2.5英寸驱动器      |
| **NVLink**   | NVIDIA® NVLINK™ v3.0                    |
| **NVSwitch** | NVIDIA® NVSwitch™                       |
| **PCIe扩展** | 丰富的PCIe 4.0接口扩展                  |
| **通信技术** | GPU之间采用NVLink和NVSwitch实现高效通信 |

- **NVIDIA HGX A100 8-GPU**：融合NVIDIA HGX A100 8-GPU架构

- **双AMD EPYC处理器**：双路AMD EPYC 7003/7002系列处理器
- **丰富的存储选项**：支持至多10个U.2 NVMe 2.5英寸驱动器
- **PCIe 4.0扩展**：提供丰富的PCIe 4.0接口扩展，满足高性能模型计算需求

- **高性能计算**：满足高性能模型计算需求
