# BC445G3-P

## 产品介绍

4U高性能计算型GPU服务器，专为AI和深度学习任务优化，配备双路AMD EPYC™ 7002/7003系列处理器；支持高达8个双宽GPU，提供强大的计算性能；配备8TB ECC DDR4内存、高达24个热插拔驱动器槽和冗余电源，确保卓越性能和可靠性；利用直连技术和PCIe 4.0插槽，实现对需求繁重工作负载的高速数据处理。

## 详细规格

| 规格项目        | 详细参数                                                              |
| --------------- | --------------------------------------------------------------------- |
| **Form Factor** | 4U Rackmount                                                          |
| **Processor**   | AMD EPYC™ 7002/7003系列处理器                                         |
| **Socket**      | Socket SP3                                                            |
| **Memory**      | 高达8TB ECC DDR4 内存                                                 |
| **Memory Type** | 3200MHz ECC DDR4                                                      |
| **Storage**     | 高达24个热插拔驱动器槽                                                |
| **PCIe Slots**  | 10个PCIe 4.0 x16 FHHL 插槽                                            |

| **Power**       | 冗余电源                                                              |
| **Dimensions**  | 高度: 178mm (7英寸)<br>宽度: 437mm (17.2英寸)<br>深度: 737mm (29英寸) |

## 产品特点

- **双路AMD EPYC处理器**：支持AMD EPYC™ 7002/7003系列处理器，提供多核多线程性能
- **强大GPU支持**：高达8个双宽GPU，满足深度学习和AI训练需求
- **高性能内存**：8TB ECC DDR4内存，频率3200MHz，提供快速数据处理能力
- **直连技术**：CPU-GPU和GPU-GPU之间采用PCIe Gen4 x16直连，实现高速数据传输
- **丰富扩展性**：10个PCIe 4.0 x16 FHHL插槽，支持各种扩展卡
- **冗余电源**：冗余电源设计，确保高负载运行时的系统可靠性
- **海量存储**：高达24个热插拔驱动器槽，满足数据存储需求
- **4U机架式设计**：适合数据中心部署，提供良好散热性能
