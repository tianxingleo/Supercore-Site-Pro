# BC445G3-P

## 产品描述

4U高性能计算型GPU服务器，专为AI和深度学习任务优化，配备双路AMD EPYC™ 7002/7003系列处理器；支持高达8个双宽GPU，提供强大的计算性能；配备8TB ECC DDR4内存、高达24个热插拔驱动器槽和冗余电源，确保卓越性能和可靠性；利用直连技术和PCIe 4.0插槽，实现对需求繁重工作负载的高速数据处理。

## 详细规格

| 规格项        | 详细参数                           |
| ------------- | ---------------------------------- |
| **外形尺寸**  | 4U机架式                           |
| **处理器**    | 双路AMD EPYC™ 7002/7003系列处理器  |
| **GPU支持**   | 高达8个双宽GPU                     |
| **内存**      | 8TB ECC DDR4内存                   |
| **驱动器槽**  | 高达24个热插拔驱动器槽             |
| **PCIe 插槽** | 利用PCIe 4.0插槽实现高速数据传输   |
| **电源**      | 冗余电源设计，确保高负载下稳定运行 |

## 产品特点

- **双宽GPU架构**：支持高达8个双宽GPU，强大的计算性能
- **AMD EPYC处理器**：双路AMD EPYC 7002/7003系列处理器
- **大容量内存**：8TB ECC DDR4内存，满足AI训练需求
- **高速数据处理**：PCIe 4.0插槽，实现对需求繁重工作负载的高速数据处理
- **热插拔驱动器**：高达24个热插拔驱动器槽，便于维护和扩展
- **冗余电源**：配备冗余电源系统，确保高负载下稳定运行
- **AI/深度学习优化**：专为AI和深度学习任务优化
