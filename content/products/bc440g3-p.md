# BC440G3-P

## 产品描述

4U高性能计算型GPU服务器，配备双插槽P+ (LGA-4689)的第三代英特尔至强可扩展处理器，支持32个内存插槽和高达8TB的3200MHz ECC DDR4 内存容量；配备6个2.5英寸热插拔NVMe/SATA/SAS驱动器托架；搭载NVIDIA HGX A100 8-GPU 40GB/80GB、NVLink和NVSwitch技术，适用于高性能计算和人工智能处理的应用场景。

## 详细规格

| 规格项       | 详细参数                                         |
| ------------ | ------------------------------------------------ |
| **外形尺寸** | 4U机架式                                         |
| **处理器**   | 双插槽P+ (LGA-4689)第三代英特尔至强可扩展处理器  |
| **内存**     | 32个内存插槽，高达8TB的3200MHz ECC DDR4 内存容量 |
| **内存类型** | DDR4 ECC RDIMM 或 LRDIUM                         |
| **存储**     | 6个2.5英寸热插拔NVMe/SATA/SAS驱动器托架          |
| **GPU支持**  | NVIDIA HGX A100 8-GPU 40GB/80GB                  |
| **技术特性** | NVLink和NVSwitch技术                             |
| **应用场景** | 高性能计算和人工智能处理                         |

## 产品特点

- **强大的GPU性能**：融合NVIDIA HGX A100 8-GPU，搭载NVLink和NVSwitch技术
- **大容量内存**：支持32个内存插槽，高达8TB DDR4内存
- **热插拔存储**：6个2.5英寸热插拔驱动器托架，便于维护和升级
- **高性能计算**：适用于高性能计算和人工智能处理的应用场景
- **NVLink技术**：GPU之间高速互联，提升数据传输效率
