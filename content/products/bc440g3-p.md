# BC440G3-P

## 产品介绍

4U高性能计算型GPU服务器，双插槽P+ (LGA-4677)第三代英特尔至强可扩展处理器；拥有32个内存插槽，高达8TB的3200MHz ECC DDR4 内存容量支持；具备24个前热插拔2.5英寸NVMe/SATA/SAS驱动器托架，12个PCIe 4.0 x16 FHFL插槽，最多支持10个双宽或10个单宽GPU，满足深度学习计算场景。

## 详细规格

| 规格项目        | 详细参数                                                                |
| --------------- | ----------------------------------------------------------------------- |
| **Form Factor** | 4U Rackmount                                                            |
| **Processor**   | 3rd Gen Intel Xeon Scalable Processor; Support CPU TDP Up to 270W       |
| **Socket**      | 2 Socket P+ (LGA-4677)                                                  |
| **Chipset**     | Intel C621A                                                             |
| **Memory**      | 32个内存插槽<br>高达8TB：32x256GB DRAM                                  |
| **Memory Type** | 支持3200MHz ECC DDR4 RDIMM or LRDIMM                                    |
| **Storage**     | 24个2.5英寸热插拔NVMe/SATA/SAS 驱动器托架                               |
| **SATA**        | 支持SATA3(6Gb/s) RAID 0/1/5/10 with Intel VROC                          |
| **NVMe**        | 支持NVMe RAID 0/1/5/10 with Intel VROC                                  |
| **PCIe Slots**  | 12 PCIe 4.0 x16 FHFL 插槽                                               |
| **GPUs**        | 最多10个双宽或10个单宽GPU                                               |
| **Network**     | 2个PCIe 4.0 x16 插槽用于AIOM(OCP3.0) 支持 NCSI                          |
| **PSU**         | 冗余钛金级电源供应                                                      |
| **Dimensions**  | 高度: 178mm (7英寸)<br>宽度: 437mm (17.2英寸)<br>深度: 650mm (25.6英寸) |

## 产品特点

- **高性能处理器**：双插槽第三代英特尔至强可扩展处理器，支持270W TDP，提供强大计算性能
- **海量内存支持**：32个内存插槽，高达8TB DDR4 ECC内存容量，3200MHz 频率
- **丰富存储选项**：24个热插拔2.5英寸NVMe/SATA/SAS驱动器托架，灵活存储配置
- **强大GPU支持**：最多支持10个双宽或10个单宽GPU，满足深度学习和AI计算需求
- **PCIe 4.0支持**：12个PCIe 4.0 x16 FHFL插槽，提供充足扩展空间
- **网络扩展**：2个PCIe 4.0 x16插槽用于AIOM(OCP3.0)网卡，支持NCSI
- **冗余电源**：冗余钛金级电源供应，确保系统稳定运行
- **4U机架式设计**：适合数据中心部署，提供充足散热空间
