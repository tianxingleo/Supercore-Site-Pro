# BS450G3

## 产品介绍

4U通用存储型服务器，配置双插槽第三代Intel® Xeon® Scalable处理器；支持16个ECC DDR4-3200内存插槽和2个Intel® Optane™持久内存专用插槽；具备36个热插拔SAS3/SATA3硬盘槽（包括4个混合NVMe槽）、2个后置SATA槽和1个SATA/NVMe M.2插槽，提供卓越的数据存储能力。

| 规格项目        | 详细参数                                                                                 |
| --------------- | ---------------------------------------------------------------------------------------- |
| **Form Factor** | 4U机架式                                                                                 |
| **Processor**   | 第3代Intel® Xeon® Scalable处理器；支持CPU TDP最高270W                                    |
| **Socket**      | 2 Socket P+ (LGA-4677)                                                                   |
| **Chipset**     | Intel C621A                                                                              |
| **Memory**      | 16个ECC DDR4-3200内存插槽                                                                |
| **Memory Type** | 支持3200MHz ECC DDR4 RDIMM 或 LRDIMM                                                     |
| **Optane**      | 2个Intel® Optane™ 持久内存专用插槽                                                       |

## 产品特点

- **高性能计算**：搭载第3代Intel® Xeon® Scalable处理器，支持CPU TDP最高达270W
- **大容量内存**：16个ECC DDR4-3200内存插槽，灵活内存配置
- **Optane持久内存**：2个Intel® Optane™专用插槽，支持持久内存技术
- **灵活存储选项**：36个热插拔硬盘槽（包括4个混合NVMe槽），多种存储选项
- **高速扩展**：3个PCIe 4.0插槽，满足各种扩展需求
- **高速网络**：2个10GbE BaseT网络接口，提供高速数据传输
- **冗余电源**：2x1600W冗余钛金级电源，确保稳定运行
- **4U机架式设计**：优化空间利用，适合数据中心部署
