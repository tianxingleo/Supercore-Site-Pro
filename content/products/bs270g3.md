# BS270G3

## 产品介绍

2U高性能存储型服务器，搭载第三代英特尔至强可扩展处理器与3200MHz ECC DDR4内存，最大可扩展至8TB的内存容量；配备了多达24个热插拔2.5英寸NVMe/SATA/SAS驱动器槽和2个M.2 NVMe/SATA3插槽，满足高容量、高速度的存储需求；同时支持多种RAID配置，确保数据安全和可靠性

## 详细规格

| 规格项目     | 详细参数                                                           |
| ------------ | ------------------------------------------------------------------ |
| **外形规格** | 2U机架式                                                           |
| **处理器**   | 第3代Intel Xeon Scalable处理器；CPU TDP最高270W                    |
| **插槽**     | 2 Socket P+ (LGA-4189)                                             |
| **芯片组**   | Intel C621A                                                        |
| **内存**     | 最多32个DIMM插槽，最高8TB内存                                      |
| **内存类型** | 3200MHz ECC DDR4                                                   |
| **存储**     | 24个2.5英寸热插拔NVMe/SATA/SAS驱动器槽 + 2个M.2 NVMe/SATA3插槽     |
| **SATA**     | 支持SATA3(6Gb) RAID 0/1/5/10 与 Intel VROC                         |
| **NVMe**     | 支持NVMe RAID 0/1/5/10 与 Intel VROC                               |
| **PCIe插槽** | 4个PCIe 4.0 x8或x16 FHHL插槽（可选）                               |
| **网络**     | 2/4x1/10GbE BaseT或2/4x1/10GbE SFP+（可选）                        |
| **PSU**      | 2x1600W冗余钛金级电源                                              |
| **尺寸**     | 高度：89mm (3.5英寸) 宽度：437mm (17.2英寸) 深度：760mm (29.9英寸) |

## 产品特点

- 高性能计算：搭载第3代Intel Xeon Scalable处理器，支持CPU TDP最高270W
- 大容量内存：32个DIMM插槽，最高8TB内存容量
- 快速存储：24个热插拔NVMe/SATA/SAS驱动器槽，支持NVMe和SATA RAID
- 多种RAID支持：支持SATA3(6Gb)和NVMe的RAID 0/1/5/10配置
- 灵活PCIe扩展：可选的PCIe 4.0插槽配置
- 冗余电源：2x1600W冗余电源，确保系统稳定
- 紧凑设计：2U机架式设计，节省数据中心空间
