# BC220G3

## 产品简介

2U通用计算型服务器，搭载第3代Intel Xeon Scalable处理器；支持高达4TB内存，拥有16个DIMM插槽，兼容3200MHz ECC DDR4 RDIMM或LRDIMM；配备8个3.5英寸热插拔SATA/SAS驱动器插槽和1个M.2 NVMe插槽；包含2个PCIe 4.0 x8 LP插槽和4个PCIe 4.0 x16 LP插槽，满足高速数据传输和扩展需求

## 详细规格

| 规格项         | 详细说明                                                                         |
| -------------- | -------------------------------------------------------------------------------- |
| **外形规格**   | 2U机架式                                                                         |
| **处理器**     | 第3代Intel Xeon Scalable处理器；支持CPU TDP高达270W                              |
| **插槽**       | 2 Socket P+ (LGA-4677)                                                           |
| **芯片组**     | Intel C621                                                                       |
| **内存**       | 最多16个DIMM插槽<br>最高4TB：16x256GB DRAM/PMem                                  |
| **内存类型**   | 支持3200MHz ECC DDR4 RDIMM或LRDIMM<br>支持Intel Optane Persistent Memory 200系列 |
| **存储**       | 8个3.5英寸热插拔SATA/SAS驱动器插槽                                               |
| **存储 (M.2)** | 1个M.2 NVMe                                                                      |
| **SATA**       | 支持SATA3(6Gb/s) RAID 0/1/5/10，配合Intel VROC                                   |
| **NVMe**       | 支持NVMe RAID 0/1/5/10，配合Intel VROC                                           |
| **PCIe 插槽**  | 2个PCIe 4.0 x8 LP 插槽<br>4个PCIe 4.0 x16 LP插槽                                 |
| **网络**       | 2个1GbE端口                                                                      |
| **电源**       | 2个1200W冗余钛金级电源                                                           |
| **尺寸**       | 高度：89毫米 (3.5英寸)<br>宽度：437毫米 (17.2英寸)<br>深度：647毫米 (25.5英寸)   |

## 产品特点

- **高性能计算**：搭载第3代Intel Xeon Scalable处理器，支持高达270W的CPU TDP
- **大容量内存**：支持最高4TB内存容量，16个DIMM插槽提供灵活的内存配置选项
- **多样化存储**：8个3.5英寸热插拔SATA/SAS驱动器插槽，配合M.2 NVMe插槽，满足不同存储需求
- **高速扩展**：提供2个PCIe 4.0 x8 LP插槽和4个PCIe 4.0 x16 LP插槽，满足高速数据传输和扩展需求
- **灵活RAID支持**：支持SATA和NVMe的RAID 0/1/5/10配置，配合Intel VROC技术
- **冗余电源**：双1200W冗余钛金级电源，确保系统稳定运行
- **紧凑设计**：2U机架式设计，节省数据中心空间
