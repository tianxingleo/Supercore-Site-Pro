# BC440G3-T

## 产品描述

4U高性能计算型塔式GPU服务器，搭载双路3rd Gen Intel Xeon Scalable C621A芯片组，支持16个DIMM槽位共计4TB 3200MHz ECC DDR4内存容量；具备丰富的PCIe 4.0扩展槽位；8个3.5英寸热插拔NVMe/SATA/SAS驱动器托架；至多支持4个NVIDIA A100、A40或RTX A6000 GPU，满足大规模数据处理以及深度学习能力。

## 详细规格

| 规格项        | 详细参数                                       |
| ------------- | ---------------------------------------------- |
| **外形尺寸**  | 4U塔式                                         |
| **处理器**    | 双路3rd Gen Intel Xeon Scalable C621A芯片组    |
| **内存**      | 16个DIMM槽位，共计4TB 3200MHz ECC DDR4内存容量 |
| **内存类型**  | DDR4 ECC RDIMM 或 LRDIUM                       |
| **存储**      | 8个3.5英寸热插拔NVMe/SATA/SAS驱动器托架        |
| **PCIe 插槽** | 丰富的PCIe 4.0扩展槽位                         |
| **GPU支持**   | 至多支持4个NVIDIA A100、A40或RTX A6000 GPU     |

## 产品特点

- **强大GPU支持**：至多支持4个NVIDIA A100、A40或RTX A6000 GPU
- **大容量内存**：16个DIMM槽位，共计4TB DDR4内存
- **塔式设计**：4U塔式设计，节省数据中心空间
- **PCIe扩展**：丰富的PCIe 4.0扩展槽位
- **热插拔存储**：8个3.5英寸热插拔驱动器托架
- **深度学习优化**：满足大规模数据处理和深度学习需求
